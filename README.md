# Algorithm

서원대학교 2학년 **알고리즘** 강의자료를 단원별로 정리한 저장소입니다.  
알고리즘 설계 기법과 복잡도 관점에서 분할 정복, 그리디, 동적 계획, 정렬, NP-완전 문제, 근사 알고리즘을 학습합니다.

---

## ✨ 핵심 내용

| 분류 | 내용 |
| --- | --- |
| 기본 개념 | 알고리즘 분석, 시간 복잡도, 문제 해결 전략 |
| 설계 기법 | 분할 정복, 그리디, 동적 계획 |
| 주요 주제 | 정렬, NP-완전 문제, 근사 알고리즘 |
| 자료 형태 | 단원별 PDF 강의자료 |

---
